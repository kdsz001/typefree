看它怎么工作
常见问题

提前回答最容易误解的几个问题。

和系统自带的听写有什么区别？

系统听写更像逐字转录，"嗯、那个"也会照打。Typefree 会在识别后整理语言——去口水词、理顺句子、规范数字标点，让输出可以直接发送或当作 Prompt。

它是完全离线的吗？

不是。当前主要路径会调用你配置的云端识别和大模型服务。它强调的是本地优先、不走我们自建的中转——而不是"断网也能用"。

支持哪些服务商和模型？

识别用火山引擎 BigASR；润色支持通义千问、智谱 GLM、豆包，可随时切换，也可以选择"不润色、直接输出原文"。

怎么收费？可以先免费试用吗？

可以。免费版每天能输入 1000 字，足够日常体验；需要无限输入再一次性买断 $19，永久使用、不是订阅。买断后 30 天内不满意可全额退款。

不太会配 API 的人能用吗？

目前更适合愿意配置服务商凭证的用户。引导里有申请入口和填写说明（5 分钟图文教程）；如果你只想先看看效果，可以直接看这页的演示。

我的录音和数据安全吗？

录音、历史和 API 凭证默认存在你自己的设备上（macOS 在 ~/.config/voicepolish/）。请求只发往你配置的服务商，不经过我们。历史保留时长也可以自己设。
