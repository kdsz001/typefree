常見問題

提前回答最容易誤解的幾個問題。

和系統自帶的聽寫有什麼區別？

系統聽寫更像逐字轉錄，"嗯、那個"也會照打。Typefree 會在識別後整理語言——去口水詞、理順句子、規範數字標點，讓輸出可以直接傳送或當作 Prompt。

它是完全離線的嗎？

不是。當前主要路徑會呼叫你配置的雲端識別和大模型服務。它強調的是本地優先、不走我們自建的中轉——而不是"斷網也能用"。

支援哪些服務商和模型？

識別用火山引擎 BigASR；潤色支援通義千問、智譜 GLM、豆包，可隨時切換，也可以選擇"不潤色、直接輸出原文"。

怎麼收費？可以先免費試用嗎？

可以。免費版每天能輸入 1000 字，足夠日常體驗；需要無限輸入再一次性買斷 $19，永久使用、不是訂閱。買斷後 14 天內不滿意可全額退款。

不太會配 API 的人能用嗎？

目前更適合願意配置服務商憑證的使用者。引導裡有申請入口和填寫說明（5 分鐘圖文教程）；如果你只想先看看效果，可以直接看這頁的演示。

我的錄音和資料安全嗎？

錄音、歷史和 API 憑證預設存在你自己的裝置上（macOS 在 ~/.config/voicepolish/）。請求只發往你配置的服務商，不經過我們。歷史保留時長也可以自己設。
