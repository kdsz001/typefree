隐私政策

最后更新：2026 年 6 月 9 日

一句话总结

Typefree 在你自己的设备上本地运行。你的录音、转写文字、历史记录和 API 凭证，默认都保存在你的设备上——我们没有用于存储你这些内容的服务器，也不会收集它们。

我们是谁

Typefree 是由 Typefree 团队开发的语音输入工具。如有隐私相关问题，可联系 ray@typefree.app。

你的语音和文字如何被处理

  • 录音在本地采集：按住快捷键时音频在你的设备上录制，松手后才进入处理。
  • 识别与整理由你自己配置的第三方服务商完成：语音识别和 AI 整理调用的是你自己申请、自己填入的第三方服务（如火山引擎、阿里云百炼等），使用你自己的 API Key，请求由你的设备直接发往这些服务商，不经过 Typefree 的服务器。这部分内容的处理受对应服务商各自隐私政策约束，建议你一并查阅。
  • 历史与凭证留在本地：识别历史、原始音频（可选保留）和 API 凭证默认保存在你设备的本地目录（macOS 位于 ~/.config/voicepolish/）。是否保留、保留多久由你设置。

购买与激活

购买通过我们的支付服务商 Paddle（Paddle.com Market Ltd，作为 Merchant of Record / 经销商）处理。你的支付信息由 Paddle 按其隐私政策收集与处理，我们不接触你的完整支付卡信息。激活买断版时，App 仅传输你的激活码本身用于校验，不上传你的语音或文字。

本网站

typefree.app 是托管在 GitHub Pages / Cloudflare 上的静态网站，不设置追踪型 Cookie、不接入第三方广告或行为分析。托管方可能按其各自政策保留标准的服务器访问日志（如 IP、访问时间）用于安全与运维。

你的权利

由于我们基本不持有你的个人数据，你对设备上的数据拥有完全控制权，可随时删除。涉及购买的数据可联系我们或 Paddle 处理。任何隐私请求，邮件至 ray@typefree.app。

政策变更

本政策如有更新，会在本页发布并更新顶部日期。
