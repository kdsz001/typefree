隱私政策

最後更新：2026 年 6 月 10 日

一句話總結

Typefree 在你自己的裝置上本地執行。你的錄音、轉寫文字、歷史記錄和 API 憑證，預設都儲存在你的裝置上——我們沒有用於儲存你這些內容的伺服器，也不會收集它們。

我們是誰

Typefree 是由獨立開發者 Rui Wang（王瑞）開發的語音輸入工具。如有隱私相關問題，可聯絡 ray@typefree.app。

你的語音和文字如何被處理

  • 錄音在本地採集：按住快捷鍵時音訊在你的裝置上錄製，鬆手後才進入處理。
  • 識別與整理由你自己配置的第三方服務商完成：語音識別和 AI 整理呼叫的是你自己申請、自己填入的第三方服務（如火山引擎、阿里雲百鍊等），使用你自己的 API Key，請求由你的裝置直接發往這些服務商，不經過 Typefree 的伺服器。這部分內容的處理受對應服務商各自隱私政策約束，建議你一併查閱。
  • 歷史與憑證留在本地：識別歷史、原始音訊（可選保留）和 API 憑證預設儲存在你裝置的本地目錄（macOS 位於 ~/.config/voicepolish/）。是否保留、保留多久由你設定。

購買與啟用

購買透過我們的支付服務商 Paddle（Paddle.com Market Ltd，作為 Merchant of Record / 經銷商）處理。你的支付資訊由 Paddle 按其隱私政策收集與處理，我們不接觸你的完整支付卡資訊。啟用買斷版時，App 會向我們的啟用伺服器（api.typefree.app，託管於 Cloudflare）發送：啟用碼、裝置指紋（硬體標識的單向雜湊，無法還原）、Mac 機型與 macOS 版本號，僅用於執行「一碼一台 Mac」的授權限制與售後支援（找回啟用碼 / 換機重置）。我們不收集你的電腦名稱、語音、文字或使用統計；啟用完成後，App 的日常使用不與我們的伺服器通訊。

本網站

typefree.app 是託管在 GitHub Pages / Cloudflare 上的靜態網站，不設定追蹤型 Cookie、不接入第三方廣告或行為分析。託管方可能按其各自政策保留標準的伺服器訪問日誌（如 IP、訪問時間）用於安全與運維。

你的權利

由於我們基本不持有你的個人資料，你對裝置上的資料擁有完全控制權，可隨時刪除。涉及購買的資料可聯絡我們或 Paddle 處理。任何隱私請求，郵件至 ray@typefree.app。
