先說清楚：API Key 是啥？
你可以把它理解成一把「鑰匙」——它讓 Typefree 能用你自己賬號下的 AI 服務（把話變成字、把字理順）。 所以你要做的就三件事：① 去拿這把鑰匙　② 複製它　③ 粘進 Typefree。拿鑰匙不花錢，註冊個賬號、點幾下就有了。
1

先選一個方案

Typefree 幫你幹兩件事：把話變成字（識別，必須）+ 把字理順（潤色，可選）。這兩件事可以用阿里、也可以用火山（豆包）。給你兩個推薦方案：

🏆 效果最好

豆包識別 + 阿里潤色

要拿 2 把鑰匙

識別用火山（豆包），識別效果最好，免費時長也更多（三檔共約 60 小時）；潤色用阿里。

適合：想要最好識別效果的人。

全用阿里

只拿 1 把鑰匙

識別和潤色都用阿里，一把鑰匙全搞定，最省事。識別送 10 小時免費額度。

適合：想最省事、先快速上手的人。

選好了？按你選的方案往下看：全用阿里只看第 2 步；要用豆包識別的，再加看第 3 步。

2

拿阿里的鑰匙

這把鑰匙識別和潤色都能用（同一個）。兩個方案都需要它。

  1. 開啟阿里百鍊控制檯。
    ↗ 開啟阿里百鍊
    第一次會讓你註冊 / 登入阿里雲賬號、同意一個協議。同意後會自動開通、自動送你免費額度，不用單獨申請任何東西。（控制檯＝阿里的網站後臺。）
  2. 點左側選單最下面的「API Key」。
  3. 新賬號第一次會讓你先做「實名認證」。點建立時如果彈出「您未完善個人基本資訊」，點彈框裡的「去完善」，按提示填你的真實姓名 + 身份證號（這是阿里的規定，大約 2 分鐘）。填完回到這個頁面、重新整理一下就能建立了。
  4. 點頁面正中紫色的「建立 API Key」按鈕。
  5. 彈出的框裡：業務空間選「預設業務空間」（一定要選，不選點確定會飄紅），描述可以不填，許可權保持「全部」，點「確定」。
  6. 建立成功會彈出完整鑰匙（sk- 開頭），馬上點「複製」存好！
    ⚠️ 重要：這串完整的鑰匙只顯示這一次，關掉視窗後就只能看到前後幾位了。所以一定要當場複製、存到備忘錄裡。
3

拿火山（豆包）的鑰匙

只有「豆包識別 + 阿里潤色」方案才需要這一步。全用阿里的可以跳過。

  1. 開啟火山語音控制檯。
    ↗ 開啟火山語音控制檯（新版）
    第一次會讓你註冊 / 登入火山引擎賬號。
  2. 新賬號先做個「實名認證」（火山要求實名後才能開通服務）。在「賬號管理 → 實名認證」裡選個人認證，用微信 / 抖音 APP 掃臉最快，手機掃一下臉，1 分鐘搞定。
  3. ⚠️ 關鍵一步：開通「錄音檔案識別 1.0」和「2.0」兩個服務。火山和阿里不一樣——識別服務要單獨開通（都免費送額度）。在左側「系統管理 → 開通管理」的「大模型」標籤頁裡，找到這兩個、各點一次「開通」：
    　·「錄音檔案識別 1.0」＝ App 的極速版 + 標準版（這兩檔都在它裡面）
    　·「錄音檔案識別 2.0」＝ App 的 2.0
    開通這兩個，App 三檔就都能用了。
  4. 開通後，去「API Key 管理」拿鑰匙。
    📍 這個頁面在哪？在左側導航欄往下劃到底，「系統管理」分組裡（「開通管理、用量統計」下面）就是「API Key 管理」，點它進入。 進入後，平臺已自動建好一把（名字像「預設體驗中心使用」），點鑰匙後面的「👁 眼睛」看到完整的、點複製即可；或點左上「+ 建立 API Key」建個新的。
4

把鑰匙填進 Typefree

  1. 開啟 Typefree 設定 → 模型。
  2. 「語音識別」選服務商：用阿里就選「百鍊（阿里）」，用豆包就選「火山引擎（豆包）」。
  3. 把剛才複製的鑰匙粘進 API Key 框。
  4. 點「測試連線」，看到 ✓ 就成功了！
  5. 潤色同理：「語音最佳化」選「百鍊（阿里）」。如果識別也用了阿里，這裡會自動複用同一把鑰匙，不用再填。

常見問題

找不到「API Key」選單怎麼辦？

阿里：在控制檯左側選單最下面。火山：在左側「系統管理」下面的「API Key 管理」。

免費額度能用多久？會偷偷扣錢嗎？

阿里 90 天、火山半年，到期作廢、不是永久。⚠️ 阿里額度用完後會自動扣費，建議進阿里控制檯開啟「免費額度用完即停」開關，就絕不會偷偷扣錢了。

點「測試連線」失敗？

八成是鑰匙沒複製全（前後帶了空格），或者服務商選錯了。重新複製完整鑰匙、確認選對「百鍊（阿里）」還是「火山引擎（豆包）」再試。

看不懂的詞，這裡查

API Key：一把「鑰匙」，讓 App 能用你賬號下的 AI 服務。
控制檯：服務商的網站後臺，管理你賬號的地方。
百鍊 / DashScope：阿里的 AI 平臺，識別和潤色都在它上面。
豆包語音 / 火山引擎：火山（位元組）的語音 AI，識別用它。
